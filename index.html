Tags de estrutura de um formulário

Ao criar um formulário temos algumas tags para adicionar valor semântico e estruturar melhor os campos de nosso formulário:
  • Label
    Em formulários usamos a tag Label para referenciar o campo que será preenchido. Ajuda também na facilidade de se clicar sobre o nome do campo e ser direcionado para a digitação no campo, uma forma inteligente de ajudar usuários mobile.

    Saiba mais sobre a tag Label:clicando aqui

  • Fieldset
    A tag Fieldset é usada para determinarmos os espaço que nosso campo ocupará, agrupando elementos no mesmo, facilitando na separação dos campos e posteriormente na estilização do documento.

    Saiba mais sobre a tag Fieldset:clicando aqui
